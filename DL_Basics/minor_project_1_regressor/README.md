# Neural Network Regressor

## Project Description
This project implements a Neural Network Regressor to predict electrical energy output using the Combined Cycle Power Plant dataset.

## Dataset
The dataset contains features such as:
- Temperature
- Exhaust Vacuum
- Ambient Pressure
- Relative Humidity

Target:
- Electrical Energy Output

## Files in this Folder
- `Regressor.ipynb` – Neural network regression model
- `powerplant_data.csv` – Dataset used for training and testing

## Tools Used
- Python
- PyTorch
- Jupyter Notebook
